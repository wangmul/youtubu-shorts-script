# YouTube Shorts 스크립터 가이드라인

이 문서는 youtube-shorts-scripter 에이전트를 호출할 때 반드시 지켜야 할 사용자 요구사항입니다.

## 필수 준수 사항

### 1. 쿠팡 언급 금지
- **절대 쿠팡에 대한 언급을 포함하지 말 것**
- 구매처, 판매처 언급 불필요
- "쿠팡에서 구매 가능", "쿠팡에서 확인" 등의 표현 사용 금지

### 2. 통합 스크립트 작성
- 여러 주제를 다룰 때 **개별 스크립트가 아닌 하나의 통합 스크립트로 작성**
- 예: 만화 3개 소개 → 3개 파일 (X), 1개 통합 파일 (O)

### 3. 글자 수 제한
- **약 1100자 내외** (1000-1200자 범위)
- YouTube Shorts에 적합한 길이 유지
- 60초 내외 영상에 맞는 분량

### 4. 구어체 작성
- 자연스러운 말투
- 대화하듯이 편하게
- 딱딱한 문어체 지양

### 5. 문장 길이와 말투
- **문장을 되도록이면 길게 작성**
- **말투는 꽤 단호하게**
- 짧게 끊지 말고 긴 호흡으로
- 확신에 찬 어조 사용

### 6. 말투 스타일 (중요!)
- **~데요, ~돼요, ~십시오, ~거든요 사용 금지**
- **~해, ~야, ~다, ~도록 해 사용** (직설적이고 친근한 반말 구어체)
- 예시:
  - (X) "물어보시는 분들 진짜 많은데요, 고민하실 필요 전혀 없습니다"
  - (O) "물어보는 사람들이 진짜 많아. 고민하지마"
  - (X) "읽기 시작하면 심장이 멈추질 않습니다"
  - (O) "읽기 시작하면 심장이 멈추질 않는다"
  - (X) "이것만 따라오시면 돼요"
  - (O) "이것만 따라오도록 해"

## 작성 팁
- 첫 3초에 시선을 사로잡는 훅 필수
- 핵심 내용만 간결하게
- 시청자가 액션을 취하고 싶게 만드는 마무리
